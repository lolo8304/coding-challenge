DHCP Message Format (BOOTP Packet)

FieldLengthDescription
op1 byteMessage op code: 1 = BOOTREQUEST, 2 = BOOTREPLY
htype1 byteHardware type (e.g., 1 = Ethernet)
hlen1 byteHardware address length (e.g., 6 for Ethernet)
hops1 byteClient sets to 0, used by relay agents
xid4 bytesTransaction ID
secs2 bytesSeconds elapsed since client began address acquisition
flags2 bytesFlags (bit 0 = broadcast flag)
ciaddr4 bytesClient IP address (if already bound)
yiaddr4 bytes"Your" IP address (offered by server)
siaddr4 bytesServer IP address
giaddr4 bytesRelay agent IP address
chaddr16 bytesClient hardware address
sname64 bytesOptional server host name
file128 bytesBoot file name
optionsvariableOptional parameters field (see below)
